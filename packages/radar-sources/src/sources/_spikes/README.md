# Source Spike Notes

BR05 source spikes live here until a source is promoted into a production
adapter. Spike notes are feasibility evidence, not runtime code.

## README Template

Each source directory should contain a `README.md` with:

- Source URL(s)
- Format
- Access and cost
- Sample inventory
- Field inventory
- Automation level
- Complexity
- Effort estimate
- Recommendation
- Risks

Large downloaded documents, paid exports, credentials, and private datasets are
not committed. Prefer stable public URLs and concise evidence notes.
